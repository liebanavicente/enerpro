Portal del Empleado

Aplicación desarrollada para la plantilla de ENERPRO. Centraliza nóminas, cuadrantes, turnos, vacaciones y solicitudes — accesible desde móvil, en español y catalán.

Contexto · ENERPRO

El problema en la empresa

La gestión interna de vigilantes y auxiliares dependía de canales informales. El portal digitaliza ese flujo.

Antes

  • Nóminas y cuadrantes por email o WhatsApp
  • Solicitudes de vacaciones sin trazabilidad
  • Coordinador saturado con consultas repetidas
  • Sin visión global del equipo ni cuadrante centralizado

Con el portal

  • Cada empleado de ENERPRO accede a su documentación 24/7
  • Solicitudes con estado en tiempo real
  • Coordinador aprueba en dos clics con badge de pendientes
  • Cuadrante visual del mes + dashboard de métricas

Tecnología

Stack técnico

Frontend
  • HTML + JavaScript vanilla (sin frameworks)
  • i18n ES / CA integrado
  • PWA instalable + service worker
  • Canvas nativo para gráficas dashboard
Backend
  • Supabase Auth + PostgreSQL
  • Storage para documentos PDF
  • Realtime para notificaciones
  • Deploy automático GitHub → Vercel
